# 🤖 Basic Chatbot using Python

## 📌 Project Overview
This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users by responding to predefined inputs such as greetings, basic questions, and exit commands. It demonstrates the use of fundamental Python concepts including functions, loops, conditional statements, and input/output operations.

## 🎯 Objectives
- Create a simple conversational chatbot.
- Understand user input handling in Python.
- Implement decision-making using if-elif-else statements.
- Practice loops and functions for interactive programs.

## ✨ Features
- Greets users with friendly responses.
- Answers basic questions like "How are you?".
- Introduces itself when asked.
- Provides help information.
- Exits the conversation when the user types "bye".
- Handles unknown inputs gracefully.

## 🛠️ Technologies Used
- Python 3

## 📚 Concepts Applied
- Functions
- While Loops
- Conditional Statements (if-elif-else)
- User Input and Output
- String Manipulation


## 💬 Sample Interaction

```text
You: hello
Bot: Hi! Nice to meet you.

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: My name is ChatBot.

You: bye
Bot: Goodbye! Have a great day.
```
